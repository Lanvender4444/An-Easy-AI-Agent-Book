#          An Easy AI Agent Book

此项目为本人复习AI Agent 相关时整理而来。

在披头士的歌曲《All You Need Is Love》中有几句歌词：

> Nothing  you  can  make  that  can't  be  made. /  尔所念之物皆可成，本就可成；
>
> No  one  you  can  save  that  can't  be  saved. /  尔所渡之人皆得救，其本得救；
>
> Nothing  you  can  do  but  you  can  learn  how  to  be  you  in  time. / 万物无须强求，愿岁月中，守尔本根，无随俗浮沉。
>
> It's  easy. / 万物皆易。

愿诸位一切顺利。

这段歌词后的下一句便是：

> All  you  need  is  love.
>
> All  you  need  is  love.
>
> All  you  need  is  love  love.
>
> Love  is  all  you  need.

也就是梦开始的地方：《Attention Is All You Need》这篇文章名称的灵感来源。

## 本地如何运行

```bash
# 本地全局安装 honkit / 本地安装 honkit
npm i -g honkit
npm install honkit --save-dev

# 运行
npm run serve

# 运行在localhost端口 4000
# http://localhost:4000/
```
